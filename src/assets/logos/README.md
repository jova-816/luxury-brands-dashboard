# Brand Logos

Drop logo files in this folder using these exact filenames:

- `fhi-heat.png` (or `.svg`)
- `prai.png`
- `youngblood.png`
- `unbrush.png`
- `nipnu.png`

## Tips for clean logos

- **Transparent PNG** is ideal so the logo blends with the sidebar background. If you grab a logo with a colored background baked in, the image will show that background as a rectangle.
- **SVG is best** — sharp at any size, tiny file. Most brand sites either embed SVG directly or expose it via brandfetch.com.
- **Height matters more than width.** The sidebar constrains logos to ~20px tall; widths can vary.
- For dark logos on the warm-off-white sidebar, transparent backgrounds work great. If a logo is white-on-dark (like UNbrush sometimes is), you may need to invert it or use a dark version.

## Where to find them

- Brand's own website header (right-click → Save image as...)
- [brandfetch.com](https://brandfetch.com) — search the brand, download SVG
- Press kits if a brand has one publicly available

## If you don't add logos

The dashboard falls back gracefully to brand name text + the colored accent dot. Nothing breaks.
